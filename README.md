# BareMetal Library for ATmega328P

A bare-metal C++ library for ATmega328P microcontrollers.  

## Class hierarchy


## Features

- DigitalPin: configure pins, read/write/toggle
- Led: turn on/off or blink LEDs
- Button: read state with optional debounce
- PWM/Servo: generate PWM signals
- IRReceiver: decode infrared remote codes
- ADC: analog input
- UART/SPI/I2C: communication interfaces
- EEPROM: read/write internal memory
- Utility: delays, mapping, and helper functions
